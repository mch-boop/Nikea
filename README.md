# NIKEA - Sistema de Gestión de Muebles y Servicios

## Descripción
NIKEA es una tienda de venta de muebles que ofrece servicios adicionales de montaje. 

## Tecnologías
- Java
- Ficheros JSON
- GitHub para control de versiones
